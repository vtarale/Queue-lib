# Queue-lib
queue lib for python
## usage
to import 
```python
import queue_lib.queue as queue
```
init
```python
q = Queue()
```
some functions
```python
# to add
q.add(<data>)
# to remove
# Ex
item = q.get()
# to display
q.display()
# to check if is empty
q.isempty() # returns bool
```
